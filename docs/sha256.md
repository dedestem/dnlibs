# sha256

Fully standalone!

### Example
``` rust
use dnlibs::sha256::hash_sha256;

fn main() {
    let value = b"Hello, world!";
    let hash = hash_sha256(value);

    println!("SHA-256 hash:");
    for byte in hash {
        print!("{:02x}", byte);
    }
    println!();
}
```
